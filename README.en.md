### 1. The Socratic AI Oracle (Replacing Politicians & Planners)

### 3. The Fractal Truth Engine & The Crossfire Arena
We reject corporate fact-checkers and authority bias. Deep controversies are mapped in a semantic Directed Acyclic Graph (DAG). Truth is not a grade; it is a **state of survival against refutation**.

We structure debates around **Epistemic Moves**—categorizing exactly *how* a theory is being defended or attacked (e.g. **Empirical Contradiction**, **Ad-Hoc Construct**, **Appeal to Authority**). This ensures radical transparency in scientific debate and exposes theories that are patched together by unobservable assumptions. The AI (Socrates) acts as an objective referee of formal logic, not a defender of consensus.

Claims exist in one of three **Epistemic States**: **Solid (יציבה)**—no community-validated refutations; **Contested (מעורערת)**—under direct attack from the opposing theory; **Shattered (הופרכה)**—the community has validated the refutation (Shattered claims remain in the weave as educational scars). Debates happen in the **Crossfire Arena**: two theories (e.g. Theory A vs. Theory B) clash horizontally. A claim in Theory A can directly attack a specific claim in Theory B; the community **Resonates** on counter-arguments, and when consensus validates a refutation, the targeted claim transitions to Shattered. The AI acts as a blind, objective referee of formal logic; it never invents evidence. One global **Feed the Arena** flow lets users submit transcripts—the AI extracts claims, classifies them by Epistemic Move, aligns them to a theory, and cross-matches them against the opposing theory.

**Web3 integration:** The Truth Engine is backed by an **Immutable Epistemic Ledger** on the blockchain. Once a node is anchored, its hash is stored on-chain—it cannot be silently altered or censored. Participation is **Sybil-resistant**: only humans holding a **Genesis Anchor SBT** (Soulbound Token) can contribute to Macro-Arenas, feed transcripts, or resonate on refutations—zero bots. When the community fundamentally disagrees with a ruling, they **fork** the logic branch on-chain, preserving both realities immutably. Controversial truth is never erased; it is branched, and the weave remains a Trust Machine for objective reality.

## 🚀 Getting Started (Local Development)

### Prerequisites
- Node.js (v18+)
- pnpm or yarn
- Foundry (`curl -L https://foundry.paradigm.xyz | bash`)
- Docker (for local Supabase)

### 1. Clone & Install
```bash
git clone https://github.com/YourOrg/mana-os.git
cd mana-os
pnpm install
```

### 2. Contracts (Foundry)
```bash
cd contracts
forge build
forge script script/DeployManaSkills.s.sol --rpc-url http://127.0.0.1:8545 --broadcast  # with Anvil running
```

### 3. Frontend
```bash
cd frontend
cp .env.example .env.local   # set NEXT_PUBLIC_MANA_SKILLS_ADDRESS, etc.
pnpm dev
```

### 4. Truth Engine & Epistemic Prism (Gemini)
The **Fractal Truth Engine** and **Epistemic Prism** (document deconstruction, anti–appeal-to-authority analysis) use **Google Gemini 1.5 Pro** for large-context, logically detached reasoning. To enable them:

1. Open **[Google AI Studio](https://aistudio.google.com/apikey)** and create or copy an API key.
2. Add to `frontend/.env.local`:
   ```bash
   GOOGLE_GENERATIVE_AI_API_KEY=your-key-here
   ```
3. Restart the frontend dev server. Document upload and “Analyze as document” will then use Gemini for whole-book–scale ingestion without manual chunking.

For full development phases, deployment, and mint scripts, see **[PROJECT_ROADMAP.en.md](PROJECT_ROADMAP.en.md)**.
