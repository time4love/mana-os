## 🛑 הבעיה: המטריקס של מחסור וזמן
האנושות רצה על מערכת הפעלה legacy מושחתת בשם "כלכלת פיאט". היא משתמשת במחסור מתוכנן, לולאות חוב אינסופיות וכפייה לשליטה באוכלוסייה.

יתר על כן, המהפכה התעשייתית הדביקה אותנו בבאג האולטימטיבי: **"זמן הוא כסף."** על ידי מדידת ערך אנושי ב"שעות", המערכת הישנה מענישה mastery (המרפא שמרפא בחמש דקות) ומתגמלת חוסר יעילות. אנחנו לכודים ב"עבודות בולשיט", מאולצים לנקוב בשעון רק כדי להרוויח טוקנים בדיוניים עבור הזכות להתקיים.

אנחנו לא צריכים מהפכה של אלימות. אנחנו צריכים hard fork. אנחנו צריכים למשוך את הסכמתנו ולצאת מהמערכת על ידי בניית אלטרנטיבה עמידה ומודעת לטראומה.

## ⚡ הפתרון: מערכת הפעלה מרפאת
Mana OS מאפשרת למיקרו-קהילות, אקו-כפרים ורשתות מבוזרות לנהל את עצמן, משאבים ותשתיות באמצעות **בינה מלאכותית, חוזים חכמים ודמוקרטיה ישירה**—בלי דולר אחד שעובר ידיים, ובלי לאלץ אף אחד למכסה.

אנחנו מחליפים "הון" ב-**מאנה** (אנרגיה מתחדשת), ואת "השעות" ב-**מעגלי ההדהוד**.

### 🏛 שלושת שכבות הריבונות
Mana OS מתוכננת לישויות ריבוניות ואינסופיות. היא דוחה כפייה דרך שלוש עמודות יסוד:
1. **שפע בסיסי ללא תנאי (UBA):** הישרדות אינה מותנית. אין במערכת מנגנון לשלול אוכל, מחסה או טיפול.
2. **הדהוד מבוסס משיכה:** אין "מנהלים" שמקצים "משימות". לוח הקהילה משמש מראה של צרכים. עבודה נטענת רק על ידי משיכה אנרגטית (הדהוד).
3. **חוזי נשמה דינמיים:** במקום מכסות שבועיות נוקשות, משתמשים מכריזים מעת לעת על יכולתם העונתית לאורקל (למשל "אני בשלב מנוחה/ריפוי" או "אני בשלב בנייה").

## 🧠 ארכיטקטורה ליבה ותכונות

### 1. האורקל הסוקרטי (במקום פוליטיקאים ומתכננים)
במקום לבחור פוליטיקאי, משתמשים משוחחים עם האורקל שלנו כדי להציע פרויקטים. האורקל פועל כ**ארכיטקט רוחני**—שואל בעדינות, מנחה לפתרונות aligned לטבע, ואז מפרק את ההצעה לתוכנית JSON של משאבים טבעיים ומעגלי מאנה.

### 2. הדהוד על-שרשרת ישיר (דמוקרטיה מבוזרת)
תוכנית המשאבים נחתמת on-chain. כל חבר קהילה מהדהד (מצביע) דרך חוזים חכמים. בלי נציגים, בלי לובי. קונצנזוס טהור, שקוף ובלתי ניתן לשינוי.

### 3. מעגלי מאנה ותדרי שליטה (Soulbound Tokens)
אנחנו לא עוקבים אחרי שעות. אנחנו עוקבים אחרי **מעגלי מאנה** (הדהודים). תרומות מחולקות ל**תדרים**: חומר, אנרגיה, ידע. שליטה מנוהלת במסלול RPG (שוליה עד מנחה). השלמת מעגל מפעילה **לולאת הכרת תודה** ומנפיקה/מעדכנת SBT לארנק.

### 4. Web3 ללא גז (Account Abstraction)
כי כסף לא קיים כאן, משתמשים לא משלמים "דמי גז". אנחנו משתמשים ב-**ERC-4337**. התשתית הקהילתית מפשטת את הבלוקצ'יין. משתמשים לוחצים "הדהד" או "טען קריאה" בלי לדאוג מאיזונים או seed phrases.

### 5. גלובלי מהיום הראשון (i18n ו-RTL)
התנועה הזו לא מקומית. Mana OS נבנית מתאריך השקה ראשון לתמיכה בכל שפה, עם עיצוב לוגי (Tailwind) ל-RTL כמו עברית וערבית.

## 🚀 התחלה (פיתוח מקומי)

### דרישות
- Node.js (v18+), pnpm או yarn, Foundry, Docker (ל-Supabase מקומי).

### התקנה
```bash
git clone https://github.com/YourOrg/mana-os.git
cd mana-os
pnpm install
```

חוזים: `cd contracts && forge build`. פרונט: `cd frontend && cp .env.example .env.local && pnpm dev`.
